# Mi Portafolio

Este es mi sitio web de portafolio personal, inspirado en el template de Developer X de Webflow.

## Estructura del Proyecto

- `index.html`: Página principal
- `css/style.css`: Estilos CSS
- `js/script.js`: Scripts JavaScript
- `images/`: Carpeta para imágenes

## Cómo Usar

1. Personaliza el contenido en `index.html` con tu información.
2. Reemplaza las imágenes placeholder con tus propias fotos.
3. Actualiza los enlaces sociales y de contacto.

## Despliegue en GitHub Pages

1. Crea un repositorio en GitHub llamado `tuusuario.github.io`.
2. Sube estos archivos al repositorio.
3. Ve a Settings > Pages y selecciona la rama main como fuente.

Para hacer un pull request a otra cuenta, asegúrate de tener permisos o fork el repositorio correspondiente.
